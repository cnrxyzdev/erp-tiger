# Logo Tiger ERP - SQL Reference & Analysis

A technical reference repository for **Logo Tiger / Tiger3 ERP**, one of the most widely used ERP systems in Turkey. This repo covers SQL-level analysis, transaction code mappings, reporting scripts, and real-world problem solutions.

---

## Contents

| Folder | Description |
|--------|-------------|
| `sql/` | SQL scripts and reference guides |
| `raporlar/` | Reporting queries and analysis |
| `moduller/` | Module-level documentation |
| `hata-cozumleri/` | Real-world bugs and solutions |

---

## Topics Covered

- TRCODE mappings across all major tables (Invoice, Stfiche, Clfiche, Bnfiche, Paytrans...)
- Current account (cari hesap) structure and queries
- Stock movement and inventory tracking
- Cash flow and bank transaction analysis
- E-invoice (e-fatura) integration notes
- SSRS report scripts

---

## Tech Stack

- **Database:** SQL Server
- **ERP:** Logo Tiger 3
- **Query Language:** T-SQL

---

## About

Logo Tiger is a leading ERP platform used by thousands of companies in Turkey for accounting, inventory, invoicing, and financial management. This repository documents its database structure at the SQL level — useful for developers building custom integrations, reports, or companion applications on top of Logo Tiger.
